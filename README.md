  <h1>30 Seconds Cmd Game</h1>
  <p>Description:</p>
  <p>
    This is a cmd line game based on the famous 30 seconds board game but with a slight twist. The objectives are, you roll the dice to see where you land on the board, and depending on the colour you landed on choose the respective color card, you will have 30 seconds to guess as many items as possible.
  </p>
  <p>There are three types of cards in this version of the game:</p>
  <ul>
    <li>Yellow Card: Easy items</li>
    <li>Blue Card: Easy to medium items</li>
    <li>Red Card: Medium to hard items</li>
  </ul>
  <p>Instructions:</p>
  <ol>
    <li>Make sure that the "red_card.txt" file is saved in the same folder as the program.</li>
    <li>The game will start and display a welcome message with instructions.</li>
    <li>Roll the dice to see where you land on the board. Board image attached to print.</li>
    <li>Depending on the colour you landed on choose the respective color card, you will have 30 seconds to guess as many items as possible.</li>
    <li>The program will display the items you need to name.</li>
    <li>When the 30 seconds are up the program will then return you to the beginning of the loop to continue playing.</li>
  </ol>
  <p>The game includes the following features:</p>
  <ul>
    <li>Dice roll function.</li>
    <li>Timer function: Countdown from 30 seconds.</li>
    <li>Ability to quit the game at any time by typing 'quit'.</li>
    <li>Three types of cards: yellow, blue, and red which uses 3 different methods to retrieve data.</li>
    <ul>
      <li>Yellow card: A [list] to retrieve data.</li>
      <li>Blue card: {Dictionary} with key-value pairs to retrieve data.</li>
      <li>Red card: Reading from a text file: to retrieve data.</li>
    </ul>
  </ul>
  <p>Notes:</p>
  <ul>
    <li>You can download the attached image of the game board in the repository directory and print it.</li>
    <li>You can choose your own household objects to represent each team and play.</li>
    <li>The timer function is set to 5 seconds for reduced waiting time and testing purposes. Edit to 30 to play.</li>
    <li>Have fun playing the 30 Seconds game!</li>
  </ul>
